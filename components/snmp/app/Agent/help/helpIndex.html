Ayuda del Monitor SNMP de ISABEL

Introducción:

Este manual incluye toda la documentación necesaria para usar el agente SNMP pueda hacerlo sin necesidad de conocer de antemano nada acerca de este software ni de su funcionamiento interno.


Los únicos conocimientos previos requeridos son nociones sobre como operar ISABEL y sus componentes; y estar familiarizados con los métodos y propósitos de la gestión de red.


En primer lugar se van a tratar las modificaciones que debe hacerse a una versión de ISABEL instalada para poder monitorizar los componentes de shared-display y video haciendo uso de cualquier gestor SNMP existente en el mercado. El funcionamiento del gestor debe conocerlo el usuario y queda fuera del ámbito de este texto.


Posteriormente se explica como arrancar y usar la interfaz gráfica que tiene el propio agente, de forma que no sea necesario disponer de un gestor SNMP comercial para acceder a la información de cada uno de los componentes.

1.- Arranque del Agente



2.- Monitorización con un gestor SNMP



3.- Comandos de Control



4.- Interfaz gráfica del Agente
